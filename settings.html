📖 Guide de configuration Collectioner

☁️ Étape 1 — Fichier JSON sur Google Drive

  1. Crée un fichier texte, nomme-le collection.json
  2. Colle ce contenu vide dedans :
{"version":"2.0","vinyls":[],"cds":[],"dvds":[],"consoles":[]}
  1. Mets-le sur Google Drive
  2. Clic droit → Partager → Tout le monde avec le lien → Lecteur
  3. Dans l'URL de partage, copie l'ID (la longue suite de caractères)
  4. Forme l'URL de téléchargement :
https://drive.google.com/uc?export=download&id=TON_ID_ICI
  1. Colle cette URL dans le champ "URL du fichier JSON" et sauvegarde
💡 Workflow : modifie sur PC → Exporter → remplace sur Drive → sur mobile l'app recharge automatiquement !

🎵 Étape 2 — Token Discogs (optionnel)

  1. Va sur discogs.com et crée un compte gratuit
  2. Mon profil → Paramètres → Développeurs
  3. Clique Generate Token → copie le token
  4. Colle-le dans le champ Discogs ci-dessus
💡 Facultatif ! MusicBrainz (0 token) est utilisé en priorité. Discogs = fallback si album non trouvé.

🎬 Étape 3 — Token TMDB (pour DVDs)

  1. Va sur themoviedb.org et crée un compte gratuit
  2. Paramètres → API → Demander une clé API → Usage personnel
  3. Copie le Jeton d'accès en lecture API (v4 auth) — le long token Bearer
  4. Colle-le dans le champ TMDB ci-dessus
💡 Gratuit, limite 40 requêtes/10 secondes. Largement suffisant pour un usage personnel.

🔄 Workflow quotidien

  1. L'app charge ton JSON Drive automatiquement au démarrage
  2. Tu ajoutes/modifies des éléments dans la collection
  3. Paramètres → Exporter → télécharge collection-DATE.json
  4. Sur Drive, remplace l'ancien fichier par le nouveau
  5. Sur ton autre appareil : recharge l'app → à jour !
