📖 Guide de configuration Collectioner

☁️ Étape 1 — Créer le Gist

  1. Va sur gist.github.com
  2. Nom du fichier : collection.json
  3. Colle ce contenu :
{"vinyls":[],"cds":[],"dvds":[],"consoles":[],"jeux":[]}
  1. Clique Créer un Gist secret
  2. Copie l'ID dans l'URL (longue suite de caractères)
💡 Secret = pas indexé par Google, mais accessible si quelqu'un a l'URL. Suffisant pour un usage personnel.

🔑 Étape 2 — Créer le Token GitHub

  1. Va sur github.com/settings/tokens
  2. Clique Generate new token (classic)
  3. Note : Collectioner
  4. Expiration : Aucune
  5. Coche uniquement gist
  6. Clique Generate token et copie-le immédiatement
⚠ GitHub ne te montre le token qu'une seule fois ! Note-le dans un endroit sûr.

🎵 Étape 3 — Token Discogs (optionnel)

  1. Va sur discogs.com → Paramètres → Développeurs
  2. Clique Generate Token → copie
💡 Facultatif — MusicBrainz est utilisé en priorité (gratuit, sans token).

🎬 Étape 4 — Token TMDB (pour DVDs)

  1. Va sur themoviedb.org → Paramètres → API
  2. Copie le Jeton d'accès en lecture API (v4)
💡 Gratuit, 40 req/10s. Nécessaire pour les pochettes DVD.
